# CLAUDE.md

## Purpose
This repository contains the design documentation for a Claude-powered assistant that helps an IT architect / engineering manager capture, refine, and drive initiatives from idea through closure — maintaining coherent artefacts across Confluence, Jira, and ServiceNow without manual cross-referencing. A lightweight backend assigns each initiative a stable identity (INI-NNNN) and owns the authoritative cross-system mapping, giving the assistant accurate, low-cost context on demand rather than re-reading source systems on every turn. You are helping design this system.

Agent instructions are in `framework/CLAUDE.md`. Read that file before taking any action.
